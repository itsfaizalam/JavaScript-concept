IIFE (Immediately Invoked Function Expression) Example and Notes

IIFE (Immediately Invoked Function Expression) ek aisa function hota hai jo turant define hone ke baad execute ho jata hai. Ye JavaScript mein kaafi common pattern hai taaki hum apne variables ko global scope se alag rakh sakein aur unka accidental overwrite na ho.

Basic Syntax of IIFE:

(function() {
    // Yeh function turant execute hoga
    console.log("Yeh ek IIFE hai!");
})();

Is example mein, function define hote hi turant execute ho jata hai aur console mein "Yeh ek IIFE hai!" print karta hai.

Arrow Function Syntax:

(() => {
    console.log("Yeh ek arrow function wala IIFE hai!");
})();

Arrow function ka use karke bhi hum IIFE ko likh sakte hain, jaise upar diya gaya example.

Example with Parameters:

let result = (function(a, b) {
    return a + b;
})(5, 10);

console.log(result); // Output: 15

Is example mein, humne IIFE ko 2 parameters diye hain (5 aur 10), jo turant add hoke result return karte hain. Console mein result 15 aayega.

HTML Integration Example:

Aap is tarah ke IIFE ko apne HTML pages mein script tag ke andar use kar sakte hain:

<script>
    (function() {
        console.log("Yeh ek IIFE hai!");
    })();
</script>

IIFE ka use encapsulation ke liye hota hai taaki variables global namespace ko pollute na karein. Is tarah ka pattern JavaScript mein kaafi useful hota hai jab humein local scope chahiye.
