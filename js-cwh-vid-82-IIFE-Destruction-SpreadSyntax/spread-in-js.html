JavaScript Spread Operator Notes

Definition

Spread operator (...) ka use arrays aur objects ko expand ya unme se elements ko extract karne ke liye hota hai. Iska use karne se aap arrays aur objects ko asani se manipulate kar sakte hain.

Examples

Example 1: Spread with Arrays


const array1 = [1, 2, 3];
const array2 = [4, 5, 6];

// Spread operator se do arrays ko merge karna
const mergedArray = [...array1, ...array2];
console.log(mergedArray); // Output: [1, 2, 3, 4, 5, 6]

Yahan ...array1 aur ...array2 ke through dono arrays ko merge kiya gaya hai.

Example 2: Spread with Function Arguments


const numbers = [1, 2, 3];

function add(x, y, z) {
    return x + y + z;
}

// Spread operator se array ko function arguments me pass karna
const result = add(...numbers);
console.log(result); // Output: 6

Yahan ...numbers ke through array ke elements ko individual arguments me convert kiya gaya hai.

Example 3: Spread with Objects


const obj1 = { a: 1, b: 2 };
const obj2 = { b: 3, c: 4 };

// Spread operator se objects ko merge karna
const mergedObj = { ...obj1, ...obj2 };
console.log(mergedObj); // Output: { a: 1, b: 3, c: 4 }

Yahan ...obj1 aur ...obj2 ko merge karte waqt b ki value obj2 se overwrite hoti hai.

Example 4: Cloning an Object


const original = { x: 1, y: 2 };
const clone = { ...original };

console.log(clone); // Output: { x: 1, y: 2 }
console.log(clone === original); // Output: false

Yahan original object ka clone create kiya gaya hai, lekin ye original se alag reference hai.

Summary

  • ... operator ka use arrays aur objects ko expand ya merge karne ke liye hota hai.
  • Ye function arguments me bhi asani se elements ko pass karne me help karta hai.
  • Objects ko merge karte waqt, last defined value overwrite hoti hai.
