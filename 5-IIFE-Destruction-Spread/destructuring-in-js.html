Destructuring in JavaScript

Destructuring JavaScript ka ek feature hai jo aapko arrays aur objects se values ko easily extract karne ki suvidha deta hai. Iska matlab hai ki aap kisi array ya object ke elements ya properties ko directly variables me assign kar sakte ho, bina unhe manually access kiye.

1. Array Destructuring

Array destructuring se aap array ke elements ko easily variables me assign kar sakte ho bina manually index access kiye.

Basic Example

const fruits = ['Apple', 'Banana', 'Orange'];
const [first, second] = fruits;
console.log(first, second); // Output: Apple Banana

Yaha fruits array ko destructure kiya gaya hai, first ko 'Apple' aur second ko 'Banana' assign hota hai.

Advanced Example with Rest Syntax

let [x, y, ...rest] = [1, 5, 7, 8, 9, 10];
console.log(x, y, rest); // Output: 1 5 [7, 8, 9, 10]

x me 1 aur y me 5 assign hota hai. Aur baaki elements [7, 8, 9, 10] ko rest array me collect kiya gaya hai ...rest syntax ke through.

Skipping Elements

const [ , secondFruit, , fourthFruit] = ['Apple', 'Banana', 'Orange', 'Grapes'];
console.log(secondFruit, fourthFruit); // Output: Banana Grapes

Yaha pe pehla aur teesra element skip kiya gaya aur sirf doosra aur chautha element destructure kiya gaya.

Default Values

const [a = 10, b = 20] = [5];
console.log(a, b); // Output: 5 20

Yaha pe b me default value 20 assign hoti hai kyunki array me doosra element missing hai.

2. Object Destructuring

Object destructuring ka use karke aap object ki properties ko easily variables me assign kar sakte ho.

Basic Example

const person = { name: 'John', age: 30, profession: 'Developer' };
const { name, age } = person;
console.log(name, age); // Output: John 30

Yaha person object ko destructure kiya gaya hai, jaha name aur age properties ko directly variables me assign kiya gaya.

Custom Variable Names

const user = { id: 1, username: 'Alice' };
const { username: userName } = user;
console.log(userName); // Output: Alice

Yaha username ko userName variable me assign kiya gaya hai.

Advanced Example with Ignored Properties

let myObj = {
    p: 12,
    q: 13,
    r: 14
};
let { p, q } = myObj;
console.log("Value of P and Q = " + p, q); // Output: Value of P and Q = 12 13

Yaha sirf p aur q properties ko destructure kiya gaya aur r ko ignore kiya gaya.

Nested Object Destructuring

const employee = {
  id: 1,
  personalInfo: {
    name: 'Jane',
    age: 28
  }
};
const { personalInfo: { name, age } } = employee;
console.log(name, age); // Output: Jane 28

Yaha employee object ke andar jo personalInfo object tha, uske name aur age ko destructure kiya gaya hai.

Default Values in Object Destructuring

const { x = 10, y = 20 } = { x: 5 };
console.log(x, y); // Output: 5 20

Yaha x me 5 assign hota hai jo object me available hai, aur y ko default value 20 milti hai kyunki wo object me present nahi hai.

Summary

  • Array Destructuring: Array elements ko easily variables me assign karna.
  • Object Destructuring: Object ki properties ko variables me assign karna.

Destructuring se aapka code concise aur readable ban jata hai, aur values extract karne me kaafi aasan ho jata hai, chahe wo arrays ho ya objects.
