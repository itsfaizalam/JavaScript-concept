JavaScript Scopes

1. Global Scope

Global scope mein jo variable define karte hain, unhe code ke kisi bhi part se access kiya ja sakta hai.


let globalVar = "Main hoon global variable";

function showGlobal() {
    console.log(globalVar); // Output: Main hoon global variable
}

showGlobal(); // Output: Main hoon global variable
console.log(globalVar); // Output: Main hoon global variable

2. Local Scope

Local scope mein jo variable define karte hain, wo sirf usi function ke andar accessible hote hain.


function showLocal() {
    let localVar = "Main hoon local variable";
    console.log(localVar); // Output: Main hoon local variable
}

showLocal(); // Output: Main hoon local variable
console.log(localVar); // Error: localVar is not defined

3. Block Scope

Block scope ka matlab hai ki agar aap let ya const use karte hain, toh variable sirf us block ke andar accessible hoga.


if (true) {
    let blockVar = "Main hoon block scoped variable";
    console.log(blockVar); // Output: Main hoon block scoped variable
}

console.log(blockVar); // Error: blockVar is not defined

Summary

- Global Scope: Ye variables sab jagah accessible hote hain.
- Local Scope: Ye variables sirf function ke andar accessible hote hain.
- Block Scope: Ye variables sirf block (jaise if statement) ke andar accessible hote hain.

Yeh scopes aapko variables ko manage karne mein help karte hain, jisse aapka code cleaner aur error-free rahe.
