Async and Await Keywords


Async Keywords:

async makes a function return a Promise

The keyword async before a function makes the function return a promise:

Async function JavaScript mein ek special type ki function hoti hai jo asynchronous (non-blocking) operations handle karti hai. Iska matlab yeh hai ki yeh function kisi bhi long-running operation ko background mein perform karne deti hai bina baaki code ko rokke. Jab wo operation complete ho jata hai, toh result ko handle kar sakte hain.
